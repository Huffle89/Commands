Universal Command Reference

This guide organizes essential commands by operating system, then by category. Each command includes syntax and a plain-English description.

🪟 Windows (CMD & PowerShell)

📁 File & Directory

dir – List files (CMD)

Get-ChildItem – List files (PowerShell)

mkdir folder – Create folder

del file.txt – Delete file (CMD)

Remove-Item file.txt – Delete file (PowerShell)

🌐 Networking

ipconfig – Show IP configuration

ping google.com – Test connectivity

Test-Connection google.com – Ping (PowerShell)

🔐 Security

net user – Manage users (CMD)

Get-LocalUser – Manage users (PowerShell)

icacls file.txt – View/edit file permissions

🐧 Linux (Bash)

📁 File & Directory

ls – List files

mkdir folder – Create folder

rm -rf folder – Delete folder

🌐 Networking

ip a – Show IP addresses

ping google.com – Test connectivity

traceroute google.com – Trace route to host

🔐 Security

chmod 755 file – Set file permissions

chown user:group file – Change file ownership

🍎 macOS (Zsh/Bash)

📁 File & Directory

open . – Open current folder in Finder

touch file.txt – Create file

🌐 Networking

networksetup -getinfo Wi-Fi – Show Wi-Fi info

ping google.com – Test connectivity

🔐 Security

sudo spctl --status – Check Gatekeeper status

sudo fdesetup status – Check FileVault status

🧑‍💻 Windows Server

🧰 Server Management

Get-WindowsFeature – List server roles

Install-WindowsFeature Web-Server – Install IIS

sconfig – Launch server config menu

🔐 Security

Set-ExecutionPolicy RemoteSigned – Allow script execution

New-LocalUser – Create user
