⚡ PowerShell Overlay

Command Description
Get-Process List all running processes
Get-Service List all services and their status
Start-Service <name> Start a specific service
Stop-Service <name> Stop a specific service
Set-ExecutionPolicy RemoteSigned Allow local scripts to run
Get-Help <cmdlet> Display help for a specific cmdlet
Get-Command List all available cmdlets
Get-EventLog -LogName System View system event logs
Test-Connection hostname Ping a host (like `ping`)
Get-ChildItem List files and folders (like `ls`)
Clear-Host Clear the PowerShell screen
Restart-Computer Restart the local computer
