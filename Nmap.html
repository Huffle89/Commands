Nmap

Network Scanning Reference

Basic ScansDescription
nmap 192.168.1.1Scan single host
nmap 192.168.1.0/24Scan subnet
nmap -vVerbose output
nmap -snPing scan (host discovery)
Port ScanningDescription
nmap -p 80Scan specific port
nmap -p-Scan all ports
nmap -FFast scan (top 100 ports)
nmap -PnSkip host discovery
Service & OS DetectionDescription
nmap -sVDetect service versions
nmap -ODetect operating system
nmap -AAggressive scan (OS + services)
nmap --osscan-guessGuess OS if uncertain
ScriptingDescription
nmap --script vulnRun vulnerability scripts
nmap --script defaultRun default scripts
nmap --script http-enumEnumerate web services
nmap --script smb-os-discoveryDiscover SMB OS info
Output & LoggingDescription
nmap -oN scan.txtNormal output to file
nmap -oX scan.xmlXML output
nmap -oG scan.gnmapGrepable output
nmap -vvExtra verbose output
