🔐 OpenSSL Overlay

🔑 Key Generation

CommandDescription
openssl genrsa -out key.pem 2048Generate RSA private key
openssl ecparam -genkey -name secp256k1 -out key.pemGenerate EC private key
openssl rsa -in key.pem -pubout -out pubkey.pemExtract public key from RSA key

📄 Certificate Signing & Inspection

CommandDescription
openssl req -new -key key.pem -out req.csrCreate certificate signing request (CSR)
openssl x509 -req -in req.csr -signkey key.pem -out cert.pemSelf-sign certificate
openssl x509 -in cert.pem -text -nooutInspect certificate contents
openssl verify -CAfile ca.pem cert.pemVerify certificate against CA

🔄 Format Conversion

CommandDescription
openssl x509 -in cert.pem -out cert.der -outform DERConvert PEM to DER
openssl x509 -in cert.der -inform DER -out cert.pem -outform PEMConvert DER to PEM
openssl pkcs12 -export -out cert.p12 -inkey key.pem -in cert.pemCreate PKCS#12 bundle
openssl pkcs12 -in cert.p12 -infoInspect PKCS#12 bundle

🔐 Encryption & Decryption

CommandDescription
openssl enc -aes-256-cbc -salt -in file.txt -out file.encEncrypt file with AES
openssl enc -aes-256-cbc -d -in file.enc -out file.txtDecrypt AES-encrypted file
openssl rand -hex 16Generate random hex string
openssl dgst -sha256 file.txtGenerate SHA-256 hash of file
